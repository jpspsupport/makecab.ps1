# makecab.ps1

Escapsulate the makecab.exe to compress folder into *.cab (ex. *.wsp/*.stp) file. 

## Example1
.makecab1.ps1 -folder FOLDER

## Example2
.makecab1.ps1 -folder CustomSolution -outFile CustomSolution.wsp
